Luanti-droid is a project aim to help deploy a full Luanti server
using Termux on an Android devide.

My first test device was an old Samsung A03s (SM037M) 2021
with the following specifications:

Helio P35 CPU
4GB of RAM (3.77 usable)
Running Android 13 with One UI 5.1.
